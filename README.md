# Classic Todo App with drag & drop! :pencil:

deployed app: https://todo-list-drag-drop.vercel.app/

Todo list app with the next features:
- Edit
- Delete
- Mark as done
- Darg & drop to change positions
- The list persist in memory 


# Tools

- cra
- Sass
- react-beautiful-dnd
- uuid
- react-icons

## To run the project

npm run start

## To run tests

npm run test
